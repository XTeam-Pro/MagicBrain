# MagicBrain - Spiking Neural Network Library

## Описание

**MagicBrain** - библиотека для работы со спайковыми нейронными сетями на основе ДНК со структурной пластичностью. Используется в экосистеме StudyNinja для продвинутых задач машинного обучения и обработки данных.

## Интеграция в StudyNinja-Eco

MagicBrain добавлен в экосистему StudyNinja как AI/ML сервис для:

- **Адаптивного обучения**: Моделирование процессов обучения студентов через спайковые нейронные сети
- **Обработки паттернов**: Анализ образовательных данных и поиск закономерностей
- **Нейронного моделирования**: Исследование когнитивных процессов обучения

## Установка

```bash
cd /root/StudyNinja-Eco/projects/MagicBrain
pip install -e .
```

## Основные компоненты

- **brain.py** - Основная логика спайковой нейронной сети
- **genome.py** - ДНК-подобная структура для кодирования нейронных связей
- **graph.py** - Графовые структуры для представления нейронных сетей
- **sampling.py** - Алгоритмы семплирования и обучения
- **tasks/** - Готовые задачи для обучения и тестирования
